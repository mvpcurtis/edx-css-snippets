Flexbox Column

.parent {
    display: flex;
    flex-direction: column;
}

This block of code enables flexbox to be active with the flow direction to be "column". This changes the axis and the flow of the page from left-to-right by default to top-to-bottom.

Flexbox Row

.parent {
    display: flex;
    flex-direction: row;
}

This block of code enables flexbox to be active with the flow direction to be "row". This means the content flows from left-to-right.

Flexbox Center

.parent {
    display: flex;
    justify-content: center;
    align-items: center;
}

This block of code enables flexbox to be active and completely centers to content within its parent element.

CSS Grid

.parent {
    display: grid;
}

This block of code enables grid.

CSS Grid Rows & Columns

.parent {
    display: grid;
    grid-template-rows: 1fr 1fr 1fr;
    grid-template-columns: 1fr 1fr 1fr;
}

This block of code enables grid and would display a 3x3 grid of content.

CSS Grid Template Areas

.parent {
    display: grid;
    grid-template-rows: 1fr 1fr 1fr;
    grid-template-columns: 1fr 1fr 1fr;
    grid-template-areas:
    "nav nav nav"
    "main main sidebar"
    "foot foot foot"
}
nav (child) {
    grid-area: nav;
} main (child) {
    grid-area: main;
} sidebar (child) {
    grid-area: sidebar;
} foot (child) {
    grid-area: foot;
}
